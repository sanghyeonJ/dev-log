### Hello java

```java
public class HelloJava {
    public static void main(String[] args) {
        System.out.println("hello java");
    }
}
```

##### 注意

Java言語は大文字と小文字を区別します。大文字と小文字が異なるとエラーが発生する可能性があります。

### 実行結果

```bash
hello java
```

##### public class HelloJava

- `HelloJava`をクラスと呼びます。クラス(class)の概念を学習する必要があります。
- 今は単純に`HelloJava.java`というファイルを作成したと理解すればよいです。
- ファイル名とクラス名は同じである必要があります。
- `{}`ブロックを使用してクラスの開始と終了を表します。

##### public static void main(String[] args)

- `main`メソッドと呼びます。関数、メソッドの概念を学習する必要があります。関数、メソッドは後で学習します。
- Javaは`main(String[] args)`メソッドを見つけてプログラムを開始します。
- 今は単純に`main`はプログラムの開始点であると理解すればよいです。
- `{}`ブロックを使用してメソッドの開始と終了を表します。

##### System.out.println("hello java");

- `System.out.println()` : 値をコンソールに出力する機能です。
- `"hello java"` : Javaは文字列を使用する際に`"`（二重引用符）を使用します。二重引用符の間に希望する文字列を囲めばよいです。
- `;` : Javaはセミコロンで文を区別します。文が終わったらセミコロンを必須で入れる必要があります。

### 実行プロセス

1. `HelloJava`プログラムを実行します。
2. Javaは開始点である`main()`メソッドを実行します。
3. `System.out.println("hello java")`に遭遇し、文字列`hello java`を出力します。
4. `main()`メソッドの`{}`ブロックが終わるとプログラムは終了します。

### ブロック(block)の例

```java
public class HelloJava {                      //HelloJavaクラスの範囲開始
    public static void main(String[] args) {  //main()メソッドの範囲開始
        System.out.println("hello java");
    }                                         //main()メソッドの範囲終了
}                                             //HelloJavaクラスの範囲終了
```

- ブロック(`{}`)が開始され終了するたびにインデントが適用されていることを確認できます。これはコードを簡単に区別し理解するのに役立つ良い慣習です。ブロックがネストされるたびにインデントの深さが追加されます。
- インデントは通常スペース4回を使用します。参考までに、IntelliJ IDEを使用するとキーボードの`Tab`を一度押すと自動的にスペース4回が適用されます。
- 参考までに、インデントを行わなくてもプログラムは動作します。しかし、コードを読むのに適していません。

## まとめ

Javaプログラムの基本構造と実行プロセスをまとめると以下のようになります：

#### Javaプログラムの基本構造

- **クラス (Class)**: `public class HelloJava` - ファイル名とクラス名は同じである必要があります
- **メソッド (Method)**: `public static void main(String[] args)` - プログラムの開始点
- **ブロック (Block)**: `{}` - クラスとメソッドの範囲を表します
- **文の区別**: `;`（セミコロン）- すべての文の終わりに必須

#### 核心概念

- **出力**: `System.out.println()` - コンソールに値を出力する機能
- **文字列**: `"文字列"` - 二重引用符で囲んで表現
- **インデント**: ブロックがネストされるたびにインデントでコードの可読性を向上（通常スペース4回）

#### 実行フロー

1. プログラム実行
2. `main()`メソッドを探す
3. メソッド内部のコードを実行
4. ブロックが終わると終了

#### 注意事項

- Javaは**大文字と小文字を区別**します
- ファイル名とクラス名は**必ず同じ**である必要があります
- すべての文の終わりに**セミコロン(`;`)**を付ける必要があります
- インデントは必須ではありませんが**コードの可読性のために推奨**されます
