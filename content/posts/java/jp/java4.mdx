### パッケージ

##### Var1

```java
package variable;

public class Var1 {
    public static void main(String[] args) {
        System.out.println(10);
        System.out.println(10);
        System.out.println(10);
    }
}
```

- パッケージはJavaファイルを区別するためのフォルダとして理解すればよいです。
- `variable`というパッケージを作成した場合、そのパッケージに入るJavaファイルの最初の行に`package variable;`のように所属するパッケージを宣言する必要があります。
- Javaファイルが位置するパッケージと`package variable`宣言の位置が同じである必要があります。

#### 実行結果

```bash
10
10
10
```

単純に数字10を3回出力するコードです。しかし、ここで数字10を3回出力する代わりに、数字20を3回出力するようにコードを変更してみましょう。

```java
package variable;

public class Var1 {
    public static void main(String[] args) {
        System.out.println(20);  //変更 10 -> 20
        System.out.println(20);  //変更 10 -> 20
        System.out.println(20);  //変更 10 -> 20
    }
}
```

数字10と書かれている箇所をすべて見つけて数字20に変更する必要があります。ここでは合計3回のコード変更が発生しました。
単純な例なのでコードを3回だけ変更しましたが、もし数字10を出力する部分が100個であれば、100個のコードをすべて変更する必要があります。

さらに進んで、ユーザーが数字を入力し、ユーザーが入力した数字を出力したい場合はどうすればよいでしょうか？ユーザーが入力した値は常に変わります。誰かは100を入力し、誰かは200を入力するかもしれません。（ユーザー入力は後で扱います）
結局、どこかに値を保管しておき、必要なときに値を取り出して読むことができる保存庫が必要です。簡単に例えると、データを入れることができる容器が必要です。

すべてのプログラミング言語は、このような問題を解決するために**変数(variable)**という機能を提供します。変数は名前の通り変えることができるという意味です。

#### Var2

```java
package variable;

public class Var2 {
    public static void main(String[] args) {
        int a; //変数宣言
        a = 10; //変数初期化
        System.out.println(a);
        System.out.println(a);
        System.out.println(a);
    }
}
```

#### a = 10 実行結果

```bash
10
10
10
```

今回は`a = 20`に変更して実行してみましょう。

#### Var2

```java
package variable;

public class Var2 {
    public static void main(String[] args) {
        int a;              // 変数宣言
        a = 20;             // 10 -> 20に変更
        System.out.println(a);
        System.out.println(a);
        System.out.println(a);
    }
}
```

#### a = 20 実行結果
```bash
20
20
20
```

`a`の値を変更すると出力結果がすべて一緒に変更されることを確認できます。

#### 変数宣言

`int a`
- 数字の整数(`integer`)を保管できる名前が`a`というデータ保存庫を作成します。これを変数と呼びます。
- このように変数を作成することを変数宣言と呼びます。
- これで変数`a`には数字の整数を保管できます。
- 数字の整数だけでなく、文字、小数など様々な種類の値を保存できる変数があります。まずは数字の整数を保存する`int`を覚えておきましょう。

#### 変数に値代入

`a = 10`
- Javaでは`=`は右側にある値を左側に保存するという意味です。数学で言う二つの値が等しい(`equals`)とは異なる意味です！
- 数字を保管できるデータ保存庫である変数`a`に値`10`を保存します。
- このように宣言した変数に初めて値を代入して保存することを変数初期化と呼びます。

#### 変数値読み取り

`System.out.println(a)`
- 変数に保存されている値を読み取って使用する方法は簡単です。変数名を書くだけでよいです。
- 変数`a`に`10`が入っている場合、Javaは実行時点に変数の値を読み取って使用します。したがって、次のように解釈されます。
    - `System.out.println(a)   //変数aの値を読み取る`
    - `System.out.println(10)  //aの値である10に変更、数字10を出力`
- 参考までに、変数の値は繰り返して読み取ることができます。変数の値を読み取ることで値がなくなるわけではありません。

### 変数値変更

変数は名前の通り変えることができる数です。簡単に言うと、変数`a`に保存された値をいつでも変更できるという意味です。
今回は途中で変数の値を変更してみましょう。

#### Var3

```java
package variable;

public class Var3 {
    public static void main(String[] args) {
        int a;           //変数宣言
        a = 10;          //変数初期化: a(10)
        System.out.println(a);    //10出力
        a = 50;          //変数値変更: a(10 -> 50)
        System.out.println(a);    //50出力
    }
}
```

#### 実行結果
```bash
10
50
```

変数の値が変更された後には、10の代わりに50が出力されたことを確認できます。
