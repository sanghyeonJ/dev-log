### 変数宣言

変数を宣言すると、コンピュータのメモリ空間を確保してそこにデータを保存できます。そして変数の名前を通じてそのメモリ空間にアクセスできます。簡単に言うと、データを保管できる空間を作成し、そこに名前を付与します。

#### Var1
```java
package variable;

public class Var1 {
    public static void main(String[] args) {
        int a;
        int b;
        int c,d;
    }
}
```

変数は次のように一つずつ宣言することもでき

```java
int a;
int b;
```

次のように一度に複数の変数を宣言することもできます。

```java
int c,d;
```

### 変数初期化

変数を宣言し、宣言した変数に初めて値を保存することを変数初期化と呼びます。

#### Var2
```java
package variable;

public class Var2 {
    public static void main(String[] args) {
        //1. 変数宣言、初期化をそれぞれ別々に
        int a;
        a = 1;
        System.out.println(a);

        int b = 2;             //2. 変数宣言と初期化を一度に
        System.out.println(b);

        int c = 3, d = 4;      //3. 複数の変数宣言と初期化を一度に
        System.out.println(c);
        System.out.println(d);
    }
}
```
1. 変数の宣言と初期化をそれぞれ別々に行うことができます。
2. 変数を宣言しながら同時に初期化できます。  
    `int b`を使用して変数`b`を作成し、その次にすぐ`b = 2`を使用して変数`b`に値2を保存します。
3. 複数の変数を宣言しながら初期化も同時に進めることができます。

### 変数は初期化する必要がある

#### Var3
```java
package variable;

public class Var6 {
    public static void main(String[] args) {
        int a;
        System.out.println(a); // コンパイルエラー発生
    }
}
```

初期化していない変数を読む場合、次のようなコンパイルエラーが発生します。

```bash
java: variable a might not have been initialized
```

コンピュータではメモリは複数のシステムが一緒に使用する空間です。そのため、様々な値が常に保存されます。  
変数を宣言すると、メモリ上のある空間を占有して使用します。しかし、その空間に以前どのような値があったかは誰も知りません。したがって、初期化を行わないと異常な値が出力される可能性があります。このような問題を予防するために、Javaは変数を初期化することを強制します。
