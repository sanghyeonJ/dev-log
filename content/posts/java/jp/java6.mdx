変数はデータを扱う種類によって様々な形式が存在します。  
次のコードを実行してみましょう。

```java
package variable;

public class Var1 {
    public static void main(String[] args) {
        int a = 100;             //整数
        double b = 10.5;         //実数
        boolean c = true;        //ブール型(boolean) true、falseを入力可能
        char d = 'A';            //文字一つ
        String e = "Hello Java"; //文字列、文字列を扱うための特別なタイプ
        System.out.println(a);
        System.out.println(b);
        System.out.println(c);
        System.out.println(d);
        System.out.println(e);
    }
}
```

#### 実行結果

```java
100
10.5
true
A
Hello Java
```

変数はデータを扱う種類によって様々な形式が存在します。このような形式を英語では`タイプ`(type)と呼びます。

- `int` : 整数を扱います。例) `1`、`100`、`1000`
- `double` : 実数を扱います。例) `0.2`、`1.5`、`100.121`
- `boolean` : ブール型タイプと呼びます。`true`、`false`の値のみ使用できます。主に真偽を判断する場所で使用します。
- `char` : 文字一つを扱う際に使用します。単一引用符(`'`)を使用して囲む必要があります。例) `'A'`、`'が'`
- `String` : 文字列を扱います。二重引用符を使用する必要があります。例) `"hello java"`

**参考** : `String`は最初の文字が大文字で始まる特別なタイプです。この部分は後でクラスを学ぶ必要があり、詳しく理解できます。今は文字列を扱う特別なタイプとして理解すればよいです。`String`に関する詳細な内容は別途扱います。

#### 自分のタイプに合うデータ使用

各変数は指定したタイプに合う値を使用する必要があります。例えば、次の最初の2つのコードはコンパイルエラーが発生します。

- `int a = "百円"` : 整数タイプに文字列 (X)
- `int a = "100"` : 整数タイプに文字列 (X)、これは数字100ではなく文字列`"100"`です。文字を表す二重引用符(`"`)で囲まれています。
- `int a = 100` : 整数タイプに整数100 (O)

### リテラル

コードで開発者が直接書いた`100`、`10.5`、`true`、`'A'`、`"Hello Java"`のような固定値をプログラミング用語でリテラル(literal)と呼びます。

```java
int a = 100;             //整数リテラル
double b = 10.5;         //実数リテラル
boolean c = true;        //ブール型リテラル
char d = 'A';            //文字一つリテラル
String e = "Hello Java"; //文字列リテラル
```

変数の値は変えることができますが、リテラルは開発者が直接入力した固定値です。したがって、リテラル自体は変わりません。

#### 'リテラル自体は変わらない'の意味

比喩で説明すると  
**変数** = **箱**  
**リテラル** = **箱に入れる硬貨**  
- 箱(a)の中に100円硬貨(100)を入れた後
- 200円硬貨(200)に交換して入れることは可能  
しかし、
- 100円硬貨が200円硬貨に変わったわけではない
- ただ別の硬貨を入れただけ
