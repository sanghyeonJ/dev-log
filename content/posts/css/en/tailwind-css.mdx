Tailwind CSS is a utility-first CSS framework that enables you to build fast and consistent UIs.

## What is Tailwind CSS?

Tailwind CSS uses pre-defined utility classes to style HTML elements directly. Unlike traditional CSS, you don't need to write separate CSS files.

## Basic Usage

```jsx
// Traditional CSS approach
<div className="container">
  <h1 className="title">Title</h1>
</div>

// Tailwind CSS approach
<div className="container mx-auto px-4">
  <h1 className="text-3xl font-bold text-gray-900">Title</h1>
</div>
```

## Responsive Design

Tailwind uses a mobile-first approach.

```jsx
<div
  className="
  text-sm          // Base (mobile)
  md:text-base     // Medium and up
  lg:text-lg       // Large screens
  xl:text-xl       // Extra large screens
"
>
  Responsive text
</div>
```

## Dark Mode

```jsx
<div
  className="
  bg-white dark:bg-gray-800
  text-gray-900 dark:text-white
"
>
  Dark mode support
</div>
```

## Custom Class Combinations

```jsx
// Using @apply directive
.btn-primary {
  @apply px-4 py-2 bg-blue-600 text-white rounded-lg hover:bg-blue-700;
}

// Or directly in components
const Button = ({ children }) => (
  <button className="px-4 py-2 bg-blue-600 text-white rounded-lg hover:bg-blue-700 transition-colors">
    {children}
  </button>
);
```

## Useful Utility Classes

```jsx
// Layout
<div className="flex items-center justify-between">
<div className="grid grid-cols-3 gap-4">
<div className="container mx-auto px-4">

// Spacing
<div className="p-4 m-2 space-y-4">

// Colors
<div className="bg-blue-500 text-white border-2 border-gray-300">

// Typography
<h1 className="text-4xl font-bold text-center">
<p className="text-gray-600 leading-relaxed">
```

## Using Plugins

```javascript
// tailwind.config.js
module.exports = {
  plugins: [require("@tailwindcss/forms"), require("@tailwindcss/typography")],
};
```

## Conclusion

Tailwind CSS is very useful for rapid prototyping and building consistent design systems. Utility classes allow you to style quickly without CSS files.
