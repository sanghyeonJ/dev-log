### 실행 결과

```java
    *
   ***
  *****
 *******
*********
 *******
  *****
   ***
    *
```

#### 핵심 포인트
- 위쪽은 별 개수 증가
- 아래쪽은 별 개수 감소
- 앞에 공백으로 가운데 정렬


#### 위쪽 삼각형 규칙

| 줄(i) | 공백 | 별 |
| :---: | :---: | :---: |
| 0 | 4 | 1 |
| 1 | 3 | 3 |
| 2 | 2 | 5 |
| 3 | 1 | 7 |
| 4 | 0 | 9 |

**규칙정리** (n은 위쪽 삼각형 줄 수)
- 공백 : `n - i - 1`
- 별 : `2 * i + 1`

#### 아래쪽 삼각형 규칙

| 줄(i) | 공백 | 별 |
| :---: | :---: | :---: |
| 3 | 1 | 7 |
| 2 | 2 | 5 |
| 1 | 3 | 3 |
| 0 | 4 | 1 |

**규칙정리** (n은 위쪽 삼각형 줄 수)
- 공백 : `n - i - 1`
- 별 : `2 * i + 1`
- 가운데 줄은 중복 출력 금지

### 코드
```java
int n = 5; // 다이아몬드의 반 높이 (가운데 줄을 포함한 위쪽 삼각형의 줄 수)

// 위쪽
for (int i = 0; i < n; i++) {
    // 공백
    for (int j = 0; j < n - i - 1; j++) {
        System.out.print(" ");
    }
    // 별
    for (int j = 0; j < 2 * i + 1; j++) {
        System.out.print("*");
    }
    System.out.println();
}

// 아래쪽
for (int i = n - 2; i >= 0; i--) {
    // 공백
    for (int j = 0; j < n - i - 1; j++) {
        System.out.print(" ");
    }
    // 별
    for (int j = 0; j < 2 * i + 1; j++) {
        System.out.print("*");
    }
    System.out.println();
}

```
