## 배열의 값을 오름차순으로 정렬하는 프로그램

30, 355, 24, 12, 98, 72, 5, 76, 60, 35, 54, 62, 2, 12, 35의 수치 배열을 오름차순으로 정렬하는 프로그램을 만들어보세요.  
정렬 방법은 삽입 정렬을 사용하세요.  

### 삽입정렬의 정리
- 삽입정렬은 배열의 왼쪽부분을 정렬된 영역으로 유지하면서,
- 오른쪽에서 하나씩 값을 꺼내 왼쪽의 정렬된 부분의 오른쪽부터 값을 비교한다.
- 비교하여 정렬되어있던쪽의 값이 더 크면 오른쪽으로 한칸밀어내는 과정을 꺼내온 값이 더 클때까지 반복한다.
- 꺼내온 값이 더 크면 해당자리에 가져온 값을 정렬해 넣는다.

### 1. key값과 비교할 값 설정

```java
for (int i = 1; i < arr.length; i++) {
    int key = arr[i];
    int j = i - 1;
}
```

- 첫번째 값(0번째 index)을 정렬된 상태라 가정하고 두번째 값(1번째 index)을 key로 설정한다.

### 2. key값 자리 찾기

```java
while (j >= 0 && arr[j] > key) {
    arr[j + 1] = arr[j];
    j--;
}
```

- 0번째 index까지
- j번째 인덱스와 key값을 비교하여 j번째 값이 더 크면
- 해당 값을 오른쪽으로 옮기고 다음 값을 비교한다.
- key값이 더 클때까지 반복

### 3. key값 삽입

```java
arr[j + 1] = key;
```

- key값과 비교해서 j번째가 더 작을때 j번째 다음자리에 key값을 삽입

### 전체 코드

```java
public static void main(String[] args) {

    int[] arr = {30, 355, 24, 12, 98, 72, 5, 76, 60, 35, 54, 62, 2, 12, 35};
    
    for (int i = 1; i < arr.length; i++) {
        int key = arr[i];
        int j = i - 1;

        // key보다 큰 값들을 오른쪽으로 이동
        while (j >= 0 && arr[j] > key) {
            arr[j + 1] = arr[j];
            j--;
        }

        // key 삽입
        arr[j + 1] = key;
    }

}
```
