## 배열의 값을 오름차순으로 정렬하는 프로그램

30, 355, 24, 12, 98, 72, 5, 76, 60, 35, 54, 62, 2, 12, 35의 수치 배열을 오름차순으로 정렬하는 프로그램을 만들어보세요.  
정렬 방법은 삽입 정렬을 사용하세요.  

### 삽입정렬의 개요
1. 왼쪽부분은 항상 정렬상태라고 가정
2. 오른쪽에서 하나 가져오기
3. 왼쪽과 값 비교 후 가져온 값의 자리를 찾을때까지 한칸씩 밀기
4. 만든 자리에 가져온 값 대입

### 전체 코드
```java
public static void main(String[] args) {

    int[] arr = {30, 355, 24, 12, 98, 72, 5, 76, 60, 35, 54, 62, 2, 12, 35};
    
    // 첫번째는 정렬된 상태라 가정하고 두번째 값부터 target으로 비교시작
    for(int i = 1; i < arr.length; i++) {
        int target = arr[i];  // 삽입할 값
        int j = i - 1;
        
        // 왼쪽 값이 target보다 크면 한칸씩 이동하고
        // 비교대상을 왼쪽으로 이동시켜 다음 값과 비교
        while( j >= 0 && arr[j] > target) {
            arr[j + 1] = arr[j];
            j--;
        }
        
        // target을 올바른 위치에 삽입
        arr[j+1] = target;
    }

}
```
