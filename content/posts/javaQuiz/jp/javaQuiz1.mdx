### 実行結果

```java
    *
   ***
  *****
 *******
*********
 *******
  *****
   ***
    *
```

#### 核心ポイント
- 上側は星の数が増加
- 下側は星の数が減少
- 前に空白で中央揃え

#### 上側三角形の規則

| 行(i) | 空白 | 星 |
| :---: | :---: | :---: |
| 0 | 4 | 1 |
| 1 | 3 | 3 |
| 2 | 2 | 5 |
| 3 | 1 | 7 |
| 4 | 0 | 9 |

**規則整理** (nは上側三角形の行数)
- 空白 : `n - i - 1`
- 星 : `2 * i + 1`

#### 下側三角形の規則

| 行(i) | 空白 | 星 |
| :---: | :---: | :---: |
| 3 | 1 | 7 |
| 2 | 2 | 5 |
| 1 | 3 | 3 |
| 0 | 4 | 1 |

**規則整理** (nは上側三角形の行数)
- 空白 : `n - i - 1`
- 星 : `2 * i + 1`
- 中央行は重複出力禁止

### コード
```java
int n = 5; // ダイヤモンドの半分の高さ (中央行を含む上側三角形の行数)

// 上側
for (int i = 0; i < n; i++) {
    // 空白
    for (int j = 0; j < n - i - 1; j++) {
        System.out.print(" ");
    }
    // 星
    for (int j = 0; j < 2 * i + 1; j++) {
        System.out.print("*");
    }
    System.out.println();
}

// 下側
for (int i = n - 2; i >= 0; i--) {
    // 空白
    for (int j = 0; j < n - i - 1; j++) {
        System.out.print(" ");
    }
    // 星
    for (int j = 0; j < 2 * i + 1; j++) {
        System.out.print("*");
    }
    System.out.println();
}

```
