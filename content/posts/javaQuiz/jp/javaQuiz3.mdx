## 配列の値を昇順でソートするプログラム

30, 355, 24, 12, 98, 72, 5, 76, 60, 35, 54, 62, 2, 12, 35 の数値配列を昇順でソートするプログラムを作成してください。  
ソート方法は挿入ソートを使用してください。  

### 挿入ソートの概要
1. 左側は常にソート済みと仮定する
2. 右側から1つ取り出す
3. 左側と値を比較し、取り出した値の位置が見つかるまで1つずつ右にずらす
4. 空いた位置に取り出した値を代入する

### 全体コード
```java
public static void main(String[] args) {

    int[] arr = {30, 355, 24, 12, 98, 72, 5, 76, 60, 35, 54, 62, 2, 12, 35};
    
    // 最初の1つはソート済みと仮定し、2番目の値からtargetとして比較開始
    for(int i = 1; i < arr.length; i++) {
        int target = arr[i];  // 挿入する値
        int j = i - 1;
        
        // 左の値がtargetより大きければ1つずつ右にずらし、
        // 比較対象を左に移動して次の値と比較
        while( j >= 0 && arr[j] > target) {
            arr[j + 1] = arr[j];
            j--;
        }
        
        // targetを正しい位置に挿入
        arr[j+1] = target;
    }

}
```
