## 配列の値を昇順でソートするプログラム

30, 355, 24, 12, 98, 72, 5, 76, 60, 35, 54, 62, 2, 12, 35 の数値配列を昇順でソートするプログラムを作成してください。  
ソート方法は挿入ソートを使用してください。  

### 挿入ソートの整理
- 挿入ソートは配列の左側をソート済みの領域として保ちながら、
- 右側から1つずつ値を取り出し、左側のソート済み部分の右端から比較する。
- ソート済み側の値の方が大きければ右に1つずらす処理を、取り出した値の方が大きくなるまで繰り返す。
- 取り出した値の方が大きくなったら、その位置に取り出した値を挿入する。

### 1. keyと比較する値の設定

```java
for (int i = 1; i < arr.length; i++) {
    int key = arr[i];
    int j = i - 1;
}
```

- 最初の値（0番目のindex）をソート済みと仮定し、2番目の値（1番目のindex）をkeyに設定する。

### 2. keyの位置を探す

```java
while (j >= 0 && arr[j] > key) {
    arr[j + 1] = arr[j];
    j--;
}
```

- 0番目のindexまで
- j番目のインデックスとkeyを比較し、j番目の値の方が大きければ
- その値を右に移し、次の値と比較する。
- keyの方が大きくなるまで繰り返す

### 3. keyの挿入

```java
arr[j + 1] = key;
```

- keyと比較してj番目の方が小さいとき、j番目の次の位置にkeyを挿入する

### 全体コード

```java
public static void main(String[] args) {

    int[] arr = {30, 355, 24, 12, 98, 72, 5, 76, 60, 35, 54, 62, 2, 12, 35};
    
    for (int i = 1; i < arr.length; i++) {
        int key = arr[i];
        int j = i - 1;

        // keyより大きい値を右に移動
        while (j >= 0 && arr[j] > key) {
            arr[j + 1] = arr[j];
            j--;
        }

        // keyを挿入
        arr[j + 1] = key;
    }

}
```
