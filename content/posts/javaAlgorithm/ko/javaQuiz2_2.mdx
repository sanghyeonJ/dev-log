## 양수,음수 / 짝수,홀수 구분하는 조건 분기

#### 변수선언
우선 변수를 정의하면  
입력값을 받을 x, 값을 검증할 bool_x를 선언한다.
```java
int x = 0;
boolean bool_x = true;
```

#### 반복문 처리
값을 검증 후 정수값이 아닌경우 반복되게 처리한다.
```java
do {
    // 정수값인 경우
    bool_x = false;

    // 정수값이 아닌 경우
    bool_x = true
}while(bool_x);
```
- 입력받은 값을 검증 후 bool_x에 true 혹은 false 값을 대입한다
- 그에 따라 잘못된 값을 입력한 경우 true를 대입하여 do-while 문을 반복하도록 한다.

#### 검증
검증 과정을 거친다.
```java
String s_x = sc.next();
try {
    x = Integer.parseInt(s_x);
    bool_x = false;
}catch(NumberFormatException e) {
    bool_x = true;
    System.out.print("다시 입력하세요 : ");
}
```

#### 분기처리
입력받은 값에 대한 분기처리를 진행한다.
```java
if(x > 0) {
    if(x % 2 == 0) {
        System.out.println("양의 짝수입니다.");
    }else {
        System.out.println("양의 홀수입니다.");
    }
}else if(x < 0) {
    if(x % 2 == 0) {
        System.out.println("음의 짝수입니다.");
    }else {
        System.out.println("음의 홀수입니다.");
    }
}else {
    System.out.println("0 입니다.");
}
```

### 전체 코드
```java
public static void main(String[] args) {
        
    Scanner sc = new Scanner(System.in);
    
    int x = 0;
    boolean bool_x = true;
    
    System.out.print("x 를 입력하세요 : ");
    do {
        String s_x = sc.next();
        try {
            x = Integer.parseInt(s_x);
            bool_x = false;
            
            if(x > 0) {
                if(x % 2 == 0) {
                    System.out.println("양의 짝수입니다.");
                }else {
                    System.out.println("양의 홀수입니다.");
                }
            }else if(x < 0) {
                if(x % 2 == 0) {
                    System.out.println("음의 짝수입니다.");
                }else {
                    System.out.println("음의 홀수입니다.");
                }
            }else {
                System.out.println("0 입니다.");
            }
        }catch(NumberFormatException e) {
            bool_x = true;
            System.out.print("다시 입력하세요 : ");
        }
    }while(bool_x);

}
```
