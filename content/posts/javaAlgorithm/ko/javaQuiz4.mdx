## 배열의 값을 오름차순으로 정렬하는 프로그램

{5, 10, 7, 8, 3, 1, 9, 2}의 수치 배열을 오름차순으로 정렬하는 프로그램을 만드십시오.  
정렬 방법은  퀵정렬을 사용하십시오.

### 퀵소트 정리

- 하나의 기준값(pivot)을 기준으로 배열을 기준값보다 큰값과 작은값으로 나누고
- 나뉜부분을 다시 퀵소트로 정렬하는 과정을 반복한다.
- 중요한것은 pivot의 위치가 아니라 값이다.
- pivot을 기준으로 작은값은 왼쪽, 큰값은 오른쪽으로 옮긴다.

### 1. pivot과 양쪽 포인터 설정

```java
public static void sort(int[] arr, int left, int right){
    int center = (left + right) / 2;
    int pivot = arr[center];
    int i = left;
    int j = right;
}
```

### 2. 값을 찾을때까지 포인터 이동

```java
while(i <= j) {
    while(arr[i] < pivot) i++;
    while(arr[j] > pivot) j--;
}
```
- 좌우 포인터위치가 교차되기 전까지 실행되어야 하므로 `i <= j` 가 참일 경우에만 반복
- `while(arr[i] < pivot) i++;` : 왼쪽 포인터값이 pivot보다 작으면 무시하고 포인터를 오른쪽으로 이동시키다가 큰값이 나오면 빠져나가는 while
- `while(arr[j] > pivot) j--;` : 오른쪽 포인터값이 pivot보다 크면 무시하고 포인터를 왼쪽으로 이동시키다가 작은값이 나오면 빠져나가는 while

### 3. 찾은 두 값의 자리를 교환

```java
while(i <= j) {
    while(arr[i] < pivot) i++;
    while(arr[j] > pivot) j--;
    
    if(i <= j) {
        swap(arr, i, j);
        i++;
        j--;
    }
}

public static void swap(int[] arr, int a, int b) {
    int temp = arr[a];
    arr[a] = arr[b];
    arr[b] = temp;
}
```
- swap메서드를 만들어 두 값의 자리를 교환한다.
- 교환이 완료된 각 포인터를 다음 칸으로 옮겨주고 while문에 따라 배열의 중간에 올때까지 반복되어
- 왼쪽엔 pivot보다 작은값, 오른쪽엔 pivot보다 큰값만 모이게 된다.

### 4. 재귀함수 호출

```java
public static void sort(int[] arr, int left, int right) {
    if(left >= right) return;
    ...
    sort(arr, left, j);
    sort(arr, i, right);
}
```
- 각각 왼쪽 부분과 오른쪽 부분을 다시 둘로나눈다.
- 함수 내부에서 재귀 호출을 통해 부분 배열의 크기가 1 이하가 될 때까지 이 과정을 반복한다.
- 더 이상 나눌 수 없는 상태가 되면 재귀가 종료되며, 전체 배열이 정렬된다.

### 5. 선언
```java
public static void main(String[] args) {

    int[] arr = {5, 10, 7, 8, 3, 1, 9, 2};
    
    sort(arr, 0, arr.length-1);
    
}
```
- 인덱스는 0부터 시작하므로 개수length에서 1을 뺀값을 넣어준다.

### 전체 코드
```java
public static void main(String[] args) {

    int[] arr = {5, 10, 7, 8, 3, 1, 9, 2};
    
    sort(arr, 0, arr.length-1);
    
}

public static void sort(int[] arr, int left, int right) {
    if(left >= right) return;
    
    int center = (left + right) / 2;
    int pivot = arr[center];
    int i = left;
    int j = right;
    
    while(i <= j) {
        while(arr[i] < pivot) i++;
        while(arr[j] > pivot) j--;
        
        if(i <= j) {
            swap(arr, i, j);
            i++;
            j--;
        }
    }
    
    sort(arr, left, j);
    sort(arr, i, right);
}

public static void swap(int[] arr, int a, int b) {
    int temp = arr[a];
    arr[a] = arr[b];
    arr[b] = temp;
}
```
